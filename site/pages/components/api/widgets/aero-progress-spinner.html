Fields

Name Type Default Description
width number 50 The width of the spinner in pixels.
height number 50 The height of the spinner in pixels.
thickness number 2 The thickness of the spinner in pixels.
background string "white" The color of the spinner's track.
color string "black" The color of the spinner's moving part.
cycle number 1 The duration of one spin cycle in seconds.

Methods

Name Description Parameters Return
spin Starts the spinner animation.
stop Stops the spinner animation and pauses at the current position.
