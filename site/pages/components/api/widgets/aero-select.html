Fields

Name Type Default Description
optionIndex number -1 The zero-based index of the currently selected option. Setting this property will update the displayed value and fire the `aero-select-changed` event.

Events

Name Description
aero-select-changed Fired when the selected option changes.

Name Field
option-index optionIndex
