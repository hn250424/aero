Fields

Name Type Default Description
resizerColor string "#ccc" The color of the resizer handles on hover.
minWidth string "0" The minimum width of the box.
maxWidth string "2000" The maximum width of the box.
minHeight string "0" The minimum height of the box.
maxHeight string "2000" The maximum height of the box.

Methods

Name Description Parameters Return
addTopResizer Enables the top resizer.
removeTopResizer Disables the top resizer.
addBottomResizer Enables the bottom resizer.
removeBottomResizer Disables the bottom resizer.
addLeftResizer Enables the left resizer.
removeLeftResizer Disables the left resizer.
addRightResizer Enables the right resizer.
removeRightResizer Disables the right resizer.

Events

Name Description
aero-resize-start Fired when a resize operation begins. The detail object contains `width`, `height`, and `edge`.
aero-resize Fired continuously during a resize operation. The detail object contains `width` and `height`.
aero-resize-end Fired when a resize operation ends. The detail object contains the final `width` and `height`.
