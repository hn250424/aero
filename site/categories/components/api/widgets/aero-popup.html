Static Methods

Name Description Parameters Return
alert Displays a alert notification on the screen. message: string, options: Partial<AeroPopupOptions> Promise<void>
confirm Displays a confirm notification on the screen. message: string, options: Partial<AeroPopupOptions> Promise<void>

Popup Options

Name Type Default Description
fontSize string "1rem" Font size for the popup content and buttons.
containerBorder string "1px solid lightgrey" Border style for the popup container.
containerBoxShadow string "0 4px 8px rgba(0, 0, 0, 0.2)" Box shadow for the popup container.
titleBoundaryColor string "lightgrey" Color of the border between title and message.
buttonPrimaryBackgroundColor string "#2563eb" Background color for the OK button.
buttonPrimaryColor string "white" Text color for the OK button.
buttonSecondaryBackgroundColor string "grey" Background color for the Cancel button.
buttonSecondaryColor string "white" Text color for the Cancel button.
buttonBorderRadius string "0" Border radius for both buttons.
